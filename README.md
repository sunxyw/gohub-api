# 这是什么

这是一个可以帮助你快速进行 API 开发，包含诸多通用功能的入门项目。

## 功能

- 多驱动用户认证（登录、注册），支持 Local、Firebase
- 支持弱密码检测，数据库记录多达一百万
- 基于 Casbin 的用户授权
- 用户级别的多语言支持
- 设备认证
- 防钓鱼代码
- 审查日志
- Etag 支持
- 自带跨域请求支持
